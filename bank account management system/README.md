## Overview
This is a console-based banking application written in C. It allows users to perform basic banking operations such as creating accounts, depositing funds, withdrawing money, and checking account details.

## Features
* **Account Creation:** Setup new user accounts with basic details.
* **Deposits:** Add funds to an existing account.
* **Withdrawals:** Remove funds (with balance checks).
* **Balance Inquiry:** View current account holdings.
* **User-Friendly Interface:** Interactive menu loop.

## Technologies Used
* **Language:** C
* **Editor:** Visual Studio Code
